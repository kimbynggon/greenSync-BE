# GreenSync (Backend)

디지털 트윈 기반 스마트팜 환경 데이터를 수집하고 관리하는 백엔드 시스템입니다.  
센서 데이터와 외부 API 데이터를 통합하여 농장 단위의 환경 정보를 관리할 수 있도록 설계되었습니다.

---

## Overview

- MQTT 기반 센서 데이터 수집 및 처리
- 외부 API(날씨) 데이터 수집 및 저장
- 사용자 인증 및 권한 관리
- 농장 단위 데이터 접근 제어

---

## Tech Stack

- Node.js, Express
- PostgreSQL
- MQTT
- JWT

---

## My Role

백엔드 개발을 담당하며 인증 구조와 데이터 처리 흐름을 설계했습니다.

- JWT 기반 인증/인가 구조 설계
- MQTT 메시지 수신 및 데이터 처리 로직 구현
- 외부 API 데이터 수집 및 저장 구조 구현
- 사용자 권한 분리 및 데이터 접근 제어 설계

---

## Key Points

- 실시간 데이터 수집부터 저장까지의 흐름을 직접 설계
- 인증 및 권한 구조를 서비스 요구사항에 맞게 구성
- 센서 데이터와 외부 데이터를 함께 처리하는 구조 경험
